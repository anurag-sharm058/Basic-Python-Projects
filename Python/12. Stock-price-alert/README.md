# Stock Price Alert Project

This Python projects created using the **Requests module**, **twilio module** and **API calls** concepts.  
This project monitors stock price changes in real-time using a financial market API. If the stock price increases or decreases by 1% or more, the application automatically sends an SMS alert to the user using Twilio.

---

## Projects Included
- Fetches real-time stock price data via API
- Calculates percentage change in stock price
- Sends SMS alerts when price changes ≥ ±1
- Uses Twilio for message delivery
- Secure credential management using environment variables

---

## Technologies Used
- Requests (API handling)
- twilio.rest (massage sending)
- dotenv / os (Environment variables)
- HTTP headers
